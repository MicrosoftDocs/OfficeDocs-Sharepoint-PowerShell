# Get-SPOContentSecurityPolicy

## SYNOPSIS

Returns all sources in the current **Content Security Policy** configuration.

## SYNTAX

### Default

```powershell
Get-SPOContentSecurityPolicy
```

## DESCRIPTION

Returns all sources in the current **Content Security Policy** configuration.
